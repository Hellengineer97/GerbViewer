# GerbViewer: Interactive Boardview

**GerbViewer** — это инструмент для создания интерактивной модели печатной платы на основе стандартных производственных файлов (Gerber и Excellon). Приложение превращает статические слои в «умную» карту, где можно отслеживать электрические связи.

---

## 🚀 Суть проекта

Создание веб-ориентированной среды для визуализации и анализа топологии печатных плат, логики электрических соединений.
Предполгается расширение в сторону компелкса ПО для реверс-инжениринга плат

## 🛠 Основные функции

### 1. Многослойный импорт
* Приложение принимает и обрабатывает Gerber-файлы, представляющие медные слои. 

### 2. Электрическая связность (Drill-Link)
* Анализирует точки пересечения отверстий с полигонами и дорожками на разных слоях.
* Автоматически объединяет разрозненные сегменты в целостные электрические цепи.
* Восстанавливает логику переходов сигнала между слоями (Vias).

### 3. Умная подсветка
Интерфейс в браузере позволяет взаимодействовать с платой:
* **Клик по дорожке:** подсвечивается весь путь прохождения сигнала по всем слоям одновременно.
* **Клик по отверстию:** визуализация всех подключенных к нему проводников.
* **Навигация:** удобное масштабирование и переключение видимости слоев.
